# 주식·퀀트 학습 아카이브 9장

## 첫 퀀트 전략 설계 보고서

---

## 1. 학습 목적

이번 장의 목적은 지금까지 배운 개념을 하나의 실제 퀀트 전략 구조로 연결하는 것이다.

이전 장들에서 다음 내용을 학습했다.

```text
주식 기초 용어
차트 기본 용어
재무제표 기초
핵심 재무지표
네이버증권에서 지표 확인하기
투자 성과지표
퀀트 기본 개념
백테스트 기초
```

9장에서는 이론을 넘어서 실제로 하나의 전략을 설계한다.

이번에 설계할 첫 전략은 다음과 같다.

```text
저PER + 퀄리티 필터 + 거래대금 필터 + 월간 리밸런싱 전략
```

쉽게 말하면 다음과 같은 전략이다.

```text
이익 대비 싼 종목 중에서,
적자는 제외하고,
재무가 너무 위험한 종목도 제외하고,
실제로 거래가 충분히 되는 종목만 골라,
여러 종목에 나눠 투자하고,
매월 다시 점검하는 전략
```

이 전략은 완성형 수익 전략이라기보다 퀀트 전략의 기본 구조를 배우기 위한 입문 전략이다.

---

# 2. 전략 개요

## 2.1 전략명

```text
저PER + 퀄리티 필터 전략
```

조금 더 자세히 표현하면 다음과 같다.

```text
이익 대비 저평가된 종목을 고르되,
너무 위험한 기업과 거래가 어려운 기업은 제외하는 전략
```

---

## 2.2 전략의 핵심 아이디어

전략의 핵심 아이디어는 다음이다.

```text
PER이 낮은 기업은 이익 대비 주가가 싸게 평가된 기업일 수 있다.
```

하지만 PER만 낮다고 무조건 좋은 종목은 아니다.

PER이 낮은 기업 중에는 다음과 같은 위험이 있을 수 있다.

```text
앞으로 이익이 줄어들 기업
일회성 이익으로 PER이 낮아 보이는 기업
부채가 너무 많은 기업
거래가 거의 없는 기업
밸류 트랩에 빠진 기업
```

따라서 저PER이라는 가치 팩터에 필터를 추가한다.

```text
적자기업 제외
부채가 너무 많은 기업 제외
거래대금이 너무 작은 기업 제외
```

전략의 기본 철학은 다음과 같다.

```text
싸 보이는 기업 중에서
최소한의 재무 안정성과 거래 가능성을 갖춘 종목만 고른다.
```

---

# 3. 전략 전체 구조

이 전략은 아래 순서로 구성된다.

```text
1. 유니버스 설정
2. 기본 필터 적용
3. 퀄리티 필터 적용
4. 거래대금 필터 적용
5. PER 낮은 순으로 정렬
6. 상위 N개 종목 선택
7. 동일비중 매수
8. 월 1회 리밸런싱
9. 백테스트로 성과 확인
10. 벤치마크와 비교
```

퀀트 전략에서 중요한 것은 각 단계가 명확해야 한다는 점이다. 기준이 모호하면 백테스트도 할 수 없고, 실전에서도 일관되게 실행하기 어렵다.

---

# 4. 유니버스 설정

## 4.1 유니버스란 무엇인가

유니버스는 투자 후보군 전체다.

쉽게 말하면 다음과 같다.

```text
내 전략이 종목을 고를 수 있는 전체 범위
```

---

## 4.2 이번 전략의 유니버스

이번 전략에서는 다음 유니버스를 사용한다.

```text
코스피 + 코스닥 전체 종목 중 일반 상장 종목
```

다만 실제 백테스트나 실전 운용에서는 아래 종목은 제외하는 것이 좋다.

```text
관리종목
거래정지 종목
상장폐지 예정 종목
스팩
우선주
ETF/ETN
리츠
비정상 데이터 종목
```

초보자용 단순 표현은 다음과 같다.

```text
코스피 + 코스닥 보통주
```

---

# 5. 기본 필터

기본 필터는 전략에 들어오면 안 되는 종목을 먼저 제거하는 단계다.

---

## 5.1 필터 1: PER > 0

```text
PER > 0
```

이 조건의 의미는 다음과 같다.

```text
순이익이 흑자인 기업만 남긴다.
```

PER이 음수인 경우는 보통 적자기업이다. 초보자용 저PER 전략에서는 적자기업을 제외하는 것이 좋다.

---

## 5.2 필터 2: PBR > 0

```text
PBR > 0
```

이 조건의 의미는 다음과 같다.

```text
자본잠식 등 비정상적인 재무 상태 가능성이 있는 기업을 제외한다.
```

PBR이 음수이거나 비정상적으로 계산되는 기업은 재무구조가 불안정할 수 있다.

---

## 5.3 필터 3: 영업이익 > 0

```text
최근 결산 기준 영업이익 > 0
```

이 조건의 의미는 다음과 같다.

```text
본업에서 돈을 버는 기업만 남긴다.
```

순이익은 일회성 이익으로 좋아 보일 수 있다. 따라서 영업이익 흑자 조건을 넣으면 본업이 최소한 살아 있는 기업을 고를 수 있다.

---

# 6. 퀄리티 필터

퀄리티 필터는 싸지만 위험한 기업을 줄이기 위한 조건이다.

---

## 6.1 필터 4: 부채비율 200% 이하

```text
부채비율 <= 200%
```

이 조건의 의미는 다음과 같다.

```text
자본 대비 부채가 너무 많은 기업은 제외한다.
```

예시:

```text
자본: 100억 원
부채: 300억 원
부채비율: 300%
```

이런 기업은 금리 상승이나 실적 악화 시 위험할 수 있다.

처음 전략에서는 부채비율을 너무 엄격하게 잡기보다 200% 또는 300% 이하 정도로 시작할 수 있다.

이번 전략의 기본값은 다음이다.

```text
부채비율 <= 200%
```

---

## 6.2 필터 5: ROE > 0

```text
ROE > 0
```

이 조건의 의미는 다음과 같다.

```text
자본을 가지고 최소한 이익을 내는 기업만 남긴다.
```

더 강한 조건을 사용하려면 아래와 같이 설정할 수도 있다.

```text
ROE >= 5%
ROE >= 10%
```

하지만 처음부터 너무 강한 조건을 사용하면 종목 수가 줄어들고 과최적화 가능성이 커질 수 있다.

초보자 기본값은 다음 정도가 적당하다.

```text
ROE > 0
```

---

# 7. 거래대금 필터

퀀트 전략은 백테스트에서만 좋아 보이면 안 된다. 실제로 사고팔 수 있어야 한다.

따라서 거래대금 필터가 필요하다.

---

## 7.1 필터 6: 최근 20일 평균 거래대금 50억 원 이상

```text
최근 20일 평균 거래대금 >= 50억 원
```

이 조건의 의미는 다음과 같다.

```text
최근 한 달 동안 매일 평균 50억 원 이상 거래된 종목만 남긴다.
```

거래대금이 너무 적으면 다음 문제가 생길 수 있다.

```text
원하는 가격에 못 살 수 있음
팔 때 호가가 밀릴 수 있음
백테스트보다 실제 수익률이 낮아질 수 있음
슬리피지가 커질 수 있음
```

거래대금 필터 기준 예시는 다음과 같다.

```text
소액 실습용: 최근 20일 평균 거래대금 30억 이상
보수적 기준: 최근 20일 평균 거래대금 50억 이상
더 안정적 기준: 최근 20일 평균 거래대금 100억 이상
```

이번 전략에서는 다음을 기본값으로 사용한다.

```text
최근 20일 평균 거래대금 >= 50억 원
```

---

# 8. 팩터 선정

이번 전략의 핵심 팩터는 PER이다.

```text
팩터: PER 낮은 순
```

즉, 필터를 통과한 종목 중에서 PER이 낮은 종목을 우선적으로 고른다.

예시:

| 종목 | PER |
| -- | --: |
| A  | 3.2 |
| B  | 4.5 |
| C  | 5.1 |
| D  | 6.8 |
| E  | 8.0 |

이 경우 A, B, C 순서로 우선순위가 높다.

핵심 해석은 다음과 같다.

```text
PER이 낮다
= 현재 이익 대비 주가가 싸게 평가되어 있을 가능성
```

단, 이 전략은 저PER이 항상 좋다는 의미가 아니다. 저PER이라는 가설을 데이터로 검증해보겠다는 의미다.

---

# 9. 종목 수 설정

처음 전략에서는 너무 적은 종목보다 분산이 필요하다.

추천 범위는 다음과 같다.

```text
20개 ~ 30개
```

종목 수가 너무 적으면 다음 문제가 있다.

```text
개별 종목 리스크가 커짐
한두 종목 악재에 크게 흔들림
```

종목 수가 너무 많으면 다음 문제가 있다.

```text
전략의 팩터 효과가 희석됨
관리와 리밸런싱이 복잡해짐
```

이번 전략의 기본값은 다음이다.

```text
상위 30개 종목
```

---

# 10. 비중 설정

이번 전략은 동일비중을 사용한다.

```text
비중: 동일비중
```

30개 종목이면 각 종목 비중은 다음과 같다.

```text
100% ÷ 30개 = 약 3.33%
```

즉, 각 종목에 자산의 약 3.33%씩 투자한다.

예를 들어 투자금이 300만 원이라면 다음과 같다.

```text
300만 원 ÷ 30개 = 종목당 약 10만 원
```

다만 실제 국내 주식은 1주 단위로 매수해야 하므로 정확히 3.33%씩 맞추기 어려울 수 있다. 백테스트에서는 동일비중으로 계산하고, 실전에서는 최대한 비슷하게 맞추는 방식이 현실적이다.

---

# 11. 리밸런싱 주기

이번 전략은 월 1회 리밸런싱으로 설계한다.

```text
리밸런싱: 월 1회
```

예시:

```text
매월 마지막 거래일 기준으로 종목을 선정
다음 거래일 시가 또는 종가에 매수/매도
```

실전적으로는 다음 표현이 더 적절하다.

```text
매월 마지막 거래일 장 마감 후 데이터를 확인하고,
다음 거래일에 포트폴리오를 조정한다.
```

이유는 다음과 같다.

```text
당일 데이터를 보고 당일 종가에 매수했다고 가정하면 현실성이 떨어질 수 있음
룩어헤드 편향을 줄이기 위해 다음 거래일 매수 기준이 더 안전함
```

---

# 12. 매수 규칙

매수 규칙은 다음과 같이 정리할 수 있다.

```text
매월 리밸런싱일에 아래 조건을 모두 만족하는 종목을 추린다.

1. 코스피/코스닥 보통주
2. PER > 0
3. PBR > 0
4. 영업이익 > 0
5. ROE > 0
6. 부채비율 <= 200%
7. 최근 20일 평균 거래대금 >= 50억 원

그중 PER이 낮은 순으로 30개를 선정한다.
선정된 30개 종목을 동일비중으로 매수한다.
```

이렇게 규칙을 명확히 작성해야 백테스트와 실전 적용이 가능하다.

---

# 13. 매도 규칙

매도 규칙은 단순하게 설정한다.

```text
다음 리밸런싱 시점에
기존 보유 종목이 상위 30개에서 제외되면 매도한다.
```

또는 필터에서 탈락하면 매도한다.

예시:

```text
PER이 비정상적으로 변함
영업이익 적자 전환
거래대금 기준 미달
부채비율 기준 초과
상위 30개에서 제외
```

초보자용 전략에서는 별도 손절 조건은 넣지 않는다.

이유는 다음과 같다.

```text
처음부터 손절 조건까지 넣으면 전략 구조가 복잡해짐
팩터 전략은 보통 리밸런싱 기준으로 교체하는 방식부터 이해하는 것이 좋음
```

다만 실전에서는 리스크 관리를 위해 나중에 손절 또는 시장 필터를 추가할 수 있다.

---

# 14. 백테스트 조건

백테스트 설계 기준은 아래처럼 잡는다.

```text
백테스트 기간:
최근 10년 이상

리밸런싱:
월 1회

매수 가격:
리밸런싱 다음 거래일 시가 또는 종가

매도 가격:
리밸런싱 다음 거래일 시가 또는 종가

비중:
동일비중

거래비용:
왕복 0.3%~0.5% 반영

벤치마크:
코스피 또는 코스피200
```

거래비용은 보수적으로 잡는 것이 좋다.

```text
수수료 + 세금 + 슬리피지 포함
왕복 0.3%~0.5%
```

---

# 15. 성과 평가 지표

백테스트 결과에서는 아래 지표를 확인한다.

```text
총수익률
CAGR
MDD
샤프비율
승률
월별 수익률
연도별 수익률
벤치마크 대비 초과수익
거래 횟수
회전율
```

초보자 기준 가장 중요한 것은 다음이다.

```text
CAGR
MDD
벤치마크 대비 성과
```

해석 기준:

```text
CAGR이 높다 = 장기 수익률이 좋다
MDD가 낮다 = 손실 구간이 견딜 만하다
벤치마크보다 좋다 = 시장 대비 의미가 있다
```

# 16. 전략 설계 요약표

| 항목     | 설정                             |
| ------ | ------------------------------ |
| 전략명    | 저PER + 퀄리티 필터 전략               |
| 유니버스   | 코스피 + 코스닥 보통주                  |
| 제외 대상  | 관리종목, 거래정지, ETF/ETN, 우선주, 스팩 등 |
| 기본 필터  | PER > 0, PBR > 0, 영업이익 > 0     |
| 퀄리티 필터 | ROE > 0, 부채비율 <= 200%          |
| 거래 필터  | 최근 20일 평균 거래대금 >= 50억 원        |
| 팩터     | PER 낮은 순                       |
| 종목 수   | 30개                            |
| 비중     | 동일비중                           |
| 리밸런싱   | 월 1회                           |
| 매수 시점  | 리밸런싱 다음 거래일                    |
| 매도 시점  | 다음 리밸런싱에서 제외 시                 |
| 거래비용   | 왕복 0.3%~0.5%                   |
| 벤치마크   | 코스피 또는 코스피200                  |
| 핵심 평가  | CAGR, MDD, 샤프비율, 벤치마크 대비 성과    |

# 17. 이 전략의 장점

이 전략의 장점은 다음과 같다.

```text
구조가 단순하다.
재무지표와 퀀트 개념을 연결하기 좋다.
PER, ROE, 부채비율, 거래대금을 모두 활용한다.
동일비중이라 이해하기 쉽다.
월 1회 리밸런싱이라 실전 적용이 비교적 가능하다.
```

이 전략은 퀀트를 처음 배울 때 좋은 구조다.

```text
가치 팩터 = PER
퀄리티 필터 = ROE, 영업이익, 부채비율
유동성 필터 = 거래대금
포트폴리오 = 30개 동일비중
리밸런싱 = 월 1회
성과평가 = CAGR, MDD
```

지금까지 배운 개념이 대부분 포함되어 있다.

# 18. 이 전략의 한계

이 전략은 완벽하지 않다.

한계는 다음과 같다.

```text
저PER 종목이 계속 저평가될 수 있음
밸류 트랩 가능성 있음
반도체, 금융, 경기민감주처럼 업황 영향을 크게 받을 수 있음
재무 데이터 발표 시차 문제가 있음
생존자 편향과 룩어헤드 편향을 조심해야 함
시장 전체 하락장에서는 같이 하락할 수 있음
```

특히 시장 전체가 무너지는 구간에서는 저PER 전략도 방어가 잘 되지 않을 수 있다.

나중에는 시장 필터를 추가할 수 있다.

예시:

```text
코스피가 200일 이동평균선 위에 있을 때만 투자
코스피가 200일선 아래면 현금 보유
```

이 조건은 차트 학습에서 배운 이동평균선 개념과 연결된다.

# 19. 개선 아이디어

첫 전략을 바로 복잡하게 만들면 안 된다. 하지만 이후 개선할 수 있는 방향은 있다.

---

## 19.1 개선 1: 저PBR 추가

```text
PER 낮은 순 + PBR 낮은 순을 함께 점수화
```

이렇게 하면 이익 대비 싼 종목뿐 아니라 자산 대비 싼 종목도 함께 고려할 수 있다.

---

## 19.2 개선 2: ROE 기준 강화

```text
ROE > 5%
ROE > 10%
```

ROE 기준을 높이면 자본 효율성이 더 좋은 기업을 고를 수 있다.

다만 조건을 너무 강하게 걸면 종목 수가 줄어들 수 있으므로 주의해야 한다.

---

## 19.3 개선 3: 영업이익 성장률 추가

```text
최근 3년 영업이익 증가율 > 0
```

이 조건은 단순히 현재 이익이 나는 기업이 아니라, 이익이 성장하는 기업을 찾기 위한 조건이다.

---

## 19.4 개선 4: 모멘텀 필터 추가

```text
종가 > 60일 이동평균선
최근 3개월 수익률 > 0
```

이 조건은 가격 흐름이 약한 종목을 제외하고, 시장에서 어느 정도 수급이 붙은 종목을 선택하기 위한 조건이다.

---

## 19.5 개선 5: 시장 필터 추가

```text
코스피 > 코스피 200일 이동평균선
```

시장 전체가 약한 구간에서는 현금 비중을 높이거나 매수를 제한하는 방식이다.

---

## 19.6 개선 시 주의사항

전략을 개선할 때 중요한 원칙은 다음과 같다.

```text
한 번에 조건을 많이 추가하지 않는다.
조건 하나를 추가할 때마다 성과가 왜 좋아졌는지 검증한다.
과최적화를 조심한다.
```

조건을 많이 추가할수록 과거 성과는 좋아 보일 수 있지만, 미래에 통하지 않을 가능성도 커진다.

# 22. 이번 장 핵심 결론

첫 퀀트 전략은 복잡할 필요가 없다.

중요한 것은 수익이 잘 날 것 같은 전략을 만드는 것이 아니라, 명확하고 검증 가능한 규칙을 만드는 것이다.

이번 전략의 핵심은 다음과 같다.

```text
싸다 → PER 낮음
최소한 돈은 번다 → 영업이익 흑자
자본 효율성이 있다 → ROE > 0
부채가 과도하지 않다 → 부채비율 <= 200%
실제로 매매 가능하다 → 거래대금 50억 이상
위험을 분산한다 → 30개 동일비중
주기적으로 갱신한다 → 월 1회 리밸런싱
성과를 검증한다 → CAGR, MDD, 벤치마크 비교
```

한 문장으로 정리하면 다음과 같다.

```text
첫 퀀트 전략은 이익 대비 싼 종목을 고르되,
적자·과도한 부채·거래 부진 종목을 제외하고,
여러 종목에 나눠 월 1회 리밸런싱하는 구조로 설계한다.
```
